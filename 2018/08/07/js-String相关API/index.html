js-String相关API

发表于 2018-08-07 | 分类于 javascript |

字符串常用属性

  • .length 返回字符串的长度。
  • .constructor 对构造函数 String 的引用。

字符串常用方法

  • stringName[index] 根据字符串的索引，访问某个字符。同样作用的：.charAt(index)。
  • .charCodeAt(index) 将位于某个索引的字符转换成 ASCII码。
  • String.fromCharCode(n) 与上相反，打印 ASCII 为n的字符。
  • .indexOf(a[, b]) 整数b可选。整数b表示从索引b开始查找字符串a，找到就会返回相应的索引，否则返回-1。如果不写整数b，那么默认从 index=0处查找。（b选值复杂情况，请参考 MDN）。
  • .lastIndexOf(a[, b]) 与上相反，如果写整数b，表示从索引b开始从后往前查找字符串a，找到则返回相应索引，否则返回-1。如果不写整数b，那么默认从 index=str.length 开始。
  • .toUpperCase() 把字符串全转换成大写字母。
  • .toLowerCase() 与上相反，把字符串全转换成小写字母。
  • .substring([a, b]) 两个参数都不写，从index=0截取到index=str.length，这并没什么意义。 当两个都写时，表示从索引a截取到索引b，并且截取左边不截取右边—[a, b)。如果 a<0 那么从索引0开始截取到索引b。如果 b<0 那么返回’’（空字符串）。如果 b>a 它俩会互换位置。
  • .slice([a, b]) 也是对字符串的截取。和上述不同的是，如果 a<0 则从索引 str.length+a 开始截取; 如果 b>a 会返回空字符串；
  • .split(s) 以字符串s为间隔切割字符串，返回切割后的数组。如果s是’’，则数组内部包含着字符串每个字符。
  • .concat(s1[, s2]) 将两个以上的字符串合并。注意：原来的字符串不变。

es6 对字符串的常用扩展

  • 模板字符串。在字符串中插入变量使用 ${}，不再像之前那样使用 + 拼接了。
  • .includes(s) 判断字符串中是否包含字符串s，返回 boolean。
  • .repeat(n) 将一个字符串重复写n次，并返回新的字符串。源字符串不变。

字符串api + 正则

这种字符串的查询方式比仅用原生字符串 API 查找简洁快速。常用可配合正则使用的 API 有：

  • .split(正则, 回调) 以正则匹配到的字符串来切割原来的字符串。
  • .replace(正则, 回调) 把回调的结果替换原来的字符串（主要使用回调中的参数）。
